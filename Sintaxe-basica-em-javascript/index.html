Sintaxe Basica em JavaScript

Primeiro chamou-se mocha, depois livescript e por causa da linguagem java que tava em alta passou-se a chamar-se javaScript... mas não tem nada haver com Java, é só Marketing !
Serve para criar scripts dinamicos que interagem e modificam os dados apresentados ao usuario, seja em sites da web ou em aplicativos para celular.
Exemplificando: Html é a estrutura do prédio, css, é a pintura o embelezamento do prédio e o javascript é a parte interativa, ou seja, é a energia elétrica e hidraulica que permite a interatividade dos ocupantes do prédio.
